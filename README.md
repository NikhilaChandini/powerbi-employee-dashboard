# Employee Utilization Dashboard (Power BI)

## Description

This project presents an interactive Power BI dashboard to analyze workforce performance and resource allocation. It provides insights into utilization trends, employee distribution, and key performance metrics to support data-driven decision-making.

## Features

* KPI tracking and performance metrics
* Interactive filters and drill-through analysis
* Workforce utilization insights
* Visual reporting dashboards

## Tools Used

* Power BI
* DAX
* Excel

## Files Included

* dashboard.pbix
* Dashboard screenshots
* PDF report
